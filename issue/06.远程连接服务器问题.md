##### 本地连接断开，同时保持程序在服务器后台运行
> 使用:link:[screen命令](https://www.jianshu.com/p/b24f597c0561)
```shell
# 创建"xk_wang"进程
screen -S xk_wang 
# 在该screen窗口中执行命令，使用Ctrl + a + d将进程detached
# 之后恢复进程
screen -r xk_wang
# 或先查看该screen的id，再恢复进程
screen -ls
screen -r 19666
```
