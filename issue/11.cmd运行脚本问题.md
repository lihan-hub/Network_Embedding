##### cmd运行python脚本
- 当脚本内容包含持续显示图片时，必须手动关闭图片窗口，cmd中的脚本才能继续运行，因此比较不方便
###### 解决办法1
> :link:[参考方法](http://bbs.eetop.cn/thread-382878-1-1.html)
```python
from time import sleep
# 使用plt.ion()解决问题，虽然会导致图片窗口不一定能显示出来
plt.ion()
# 只要在plt.show前加上plt.ion()就行
plt.show()
plt.close()
```
